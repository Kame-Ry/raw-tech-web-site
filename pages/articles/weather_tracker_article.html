Weather Tracker Web: A Real-Time Weather Application

The Weather Tracker Web App is a lightweight, real-time weather tracking web application that utilizes the OpenWeather API. It is designed to provide users with accurate weather updates, including current conditions, hourly trends, and a 5-day forecast, all within a sleek and responsive UI.

Project Overview

This project was developed to offer an easy-to-use weather application that fetches data securely using Vercel’s serverless functions. The interface is built with modern web technologies, ensuring smooth user experience and scalability.

Core Features

  • Search for any city’s weather using OpenWeather API.
  • Display current weather conditions (temperature, humidity, wind speed, etc.).
  • Hourly forecast graph displaying a 12-hour trend.
  • 5-day weather forecast with weather icons.
  • Secure API handling through Vercel serverless functions.
  • Responsive design built with TailwindCSS.

Setting Up Weather Tracker Web

To run the application, follow these steps:

  1. Visit the live app: Weather Tracker Web.
  2. Enter a city name in the search bar.
  3. Click "Get Weather" to retrieve the latest weather data.
  4. View current conditions, hourly trends, and a 5-day forecast.

Secure API Handling with Vercel

export default async function handler(req, res) {
    const { city } = req.query;
    const API_KEY = process.env.WEATHER_API_KEY;
    const weatherUrl = `https://api.openweathermap.org/data/2.5/weather?q=${city}&units=metric&appid=${API_KEY}`;
    const response = await fetch(weatherUrl);
    const data = await response.json();
    res.status(200).json(data);
}

Serverless functions on Vercel handle API requests securely, preventing direct exposure of API keys.

Deployment on Vercel

The project is deployed on Vercel, allowing for seamless updates and performance optimization.

  1. Fork or clone the repository.
  2. Sign in to Vercel and import your GitHub repository.
  3. Configure environment variables (e.g., OPENWEATHER_API_KEY).
  4. Deploy via the Vercel dashboard.
